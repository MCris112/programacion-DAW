# UNIDAD 5: Desarrollo de diapositivas
Repositorio con las actividades realizadas en clase durante la Unidad 5, centradas en el diseño y desarrollo de presentaciones y en la práctica de programación orientada a objetos en Java.

El objetivo es consolidar los conceptos vistos en las diapositivas de teoría mediante pequeñas prácticas guiadas, reutilizables como material de estudio.

## Contenido de las actividades
Las actividades siguen la progresión de la unidad y las diapositivas: se parte de ejercicios sencillos y se avanza hacia herencia, clases abstractas, interfaces, colecciones y manejo básico de archivos.

Ejemplos típicos de lo que cubren las actividades:
- Definición de clases con atributos, constructores, getters y setters (ejercicios de modelos simples como productos, vehículos o animales).
- Uso de herencia para especializar clases base en subclases con comportamiento específico.
- Clases abstractas y métodos abstractos para definir plantillas comunes (por ejemplo, Vehiculo, Producto, Animal).
- Implementación de interfaces para definir comportamientos (por ejemplo, interfaces como Movible, Sonoro, Volador).
- Polimorfismo con ArrayList de la clase base o de la interfaz, recorriendo listas y llamando a métodos comunes (mover(), emitirSonido(), precioFinal(), etc.).
- Pequeños ejemplos de lectura/escritura de archivos de texto para apoyar las actividades explicadas en las diapositivas.

Cada actividad está pensada para reforzar un concepto concreto, usando ejemplos cercanos a los vistos en clase y en las presentaciones de la unidad.

## Objetivos de aprendizaje
A lo largo de la Unidad 5, las actividades buscan que el estudiante:
- Comprenda y practique el diseño de clases y objetos a partir de enunciados de problemas.
- Use la herencia y el polimorfismo para reutilizar código y modelar jerarquías.
- Aplique clases abstractas e interfaces para separar qué hace un objeto de cómo lo hace.
- Trabaje con colecciones como ArrayList para almacenar y recorrer conjuntos de objetos.
- Conecte las explicaciones de las diapositivas con código real, ejecutable y fácil de ampliar.
