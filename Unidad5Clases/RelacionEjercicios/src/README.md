# Unidad 5: Relación de ejercicios

Proyecto de ejercicios en Java para la Unidad 5 de Programación, enfocado en conceptos avanzados de POO: manejo de archivos, JSON parsing, genéricos, colecciones, herencia, clases abstractas e interfaces.

## Estructura del Proyecto

```
src/
├── Activity1/ # Fusión de archivos de texto
├── Activity2/ # Filtrado JSON de superhéroes DC
├── Activity3/ # Estadísticas de notas de alumnos
├── Activity4/ # Clase genérica Caja<T>
├── Activity5/ # Sistema de autenticación con HashMap
├── Activity6/ # Clasificación jerárquica de animales
├── Activity7/ # Clases abstractas: Vehículos
├── Activity8/ # Interfaces: Vehículos voladores
└── Utilities/ # Clases auxiliares (Table, MC, MCBox)
```


## Utilidades

### Table
Clase auxiliar para generar tablas formateadas en consola con bordes ASCII.

### MC
Clase auxiliar para títulos decorativos y líneas separadoras.

### MCBox
Clase auxiliar para cajas de texto decoradas.


# Conceptos Clave por Actividad

| Actividad | Conceptos principales                                |
| --------- | ---------------------------------------------------- |
| 1         | FileReader, FileWriter, manejo de rutas              |
| 2         | JSON parsing, JSONObject, JSONArray, filtrado        |
| 3         | JSON anidado, HashMap, cálculos estadísticos         |
| 4         | Genéricos, ArrayList<T>, métodos de colección        |
| 5         | HashMap, Scanner, validación, control de flujo       |
| 6         | Herencia multinivel, clases abstractas, polimorfismo |

# Aprendizaje Obtenido
Este proyecto cubre los fundamentos avanzados de POO en Java:

## Estructuras de datos avanzadas:
- HashMap para relaciones clave-valor
- ArrayList con genéricos
- Parsing y manipulación de JSON

## Paradigma orientado a objetos:

- Herencia simple y multinivel
- Clases abstractas como plantillas 
- Interfaces como contratos 
- Polimorfismo para flexibilidad de código
- Encapsulación con getters/setters

## Manejo de archivos:
- Lectura y escritura de archivos de texto 
- Parsing de archivos JSON 
- Gestión de rutas y nombres
